例: 0.0.260101-alpha
 - 主版本号: 在完成编辑器基础功能开发时更新为1, 在有不可向后兼容的更新时改变.             格式: Int
 - 副版本号: 在进行功能更新时改变, pre-alpha阶段为0.                                      格式: Int
 - 修订号  : 修订日期. 允许同一天多次Commit.                                              格式: YYMMDD
 - 修订类型: -alpha: 内部调试; -beta: 内测; -pre-release: (不使用)公测; -release: 发行版. 格式: Text

注: 更新日志从0.0.260101开始记录.
